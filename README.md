# excel-projects1
Excel-based salary sheet with PF, ESI, and net salary calculation
This project demonstrates an Excel-based salary sheet that includes:
- Basic salary calculation
- HRA, PF, and ESI deductions
- Gross and Net salary calculation
- Conditional formulas and percentage deductions

Tools used:
- Microsoft Excel
- IF, VLOOKUP/XLOOKUP, Percentage formulas

Status: In Progress
